# PythonTeacher

Interview prep for Python: **61 coding challenges** and **300 flashcards** across 20 topics, in a dark, focused learning UI.

```bash
git clone https://github.com/betexcr/python-teacher.git
cd python-teacher
npm install
npm run dev
```

Open http://localhost:5173

## What's included

### Python Challenges (61)

| Level | Count |
|-------|-------|
| Easy | 20 |
| Medium | 28 |
| Hard | 8 |
| Very Hard | 5 |

### Flashcards (20 decks, 300 cards)

Study URLs: `/flashcards/fundamentals`, `/flashcards/fastapi`, `/flashcards/rag`, etc.

### Other tracks

- **Python Basics** — guided tutorial from zero
- **Python Patterns** — 16 idioms and design pattern guides
- **System Design** — 21 backend/AI architecture walkthroughs

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Generate assets + start Vite |
| `npm run build` | Production build |
| `npm run generate` | Regenerate challenges, flashcards, OG images |

## Tech stack

- React 19 + TypeScript + Vite
- React Router 7
- Plain CSS with Python gold/blue theme
